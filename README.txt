Shift-Manager (not longer supported in android 6 and above)
https://www.facebook.com/ShiftManagerAp/

An android app which help the user track his shifts( very useful to waiters )

The user have to create new Work (work name ,unique color ,salary method).

(user may work in more than one work)

The user inserts shift, of the selected work he wants to track.


Allows the user view his shifts inside a beautiful calender.

Allows the user view month summery (or any other time periods) .
